# 05. 动态控制与高级特性 (专家篇)

## 1. 强制刷新与缓存控制 (`IS_CHANGED`)
ComfyUI 默认极其积极地缓存结果。如果输入参数未变，节点不会重新执行。
若你的节点输出随机数、当前时间，或读取外部变化的文件，必须实现此方法。

```python

        # 也可以返回基于参数计算的 Hash，只有特定参数变了才运行
        import time
        return float("NaN") # 最暴力的不缓存
